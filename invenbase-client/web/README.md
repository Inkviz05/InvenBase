# InvenBase - Inventory Management System

React приложение для управления оборудованием и бронированиями.

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Структура

- `src/api/` - API клиенты
- `src/components/` - React компоненты
- `src/context/` - React контексты (Auth)
- `src/pages/` - Страницы приложения
- `src/config.js` - Конфигурация

## Технологии

- React 18
- React Router 6
- Axios для HTTP запросов
- Vite для сборки
- Material Icons для иконок
- html5-qrcode для сканирования QR-кодов
