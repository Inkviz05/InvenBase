# Инструкция по сборке и использованию

## Быстрый старт

### 1. Подготовка веб-приложения

```bash
cd kvantoriym-client/web
npm install
```

Создайте файл `.env`:
```
VITE_API_URL=http://localhost:8080/api
```

Запустите dev-сервер для разработки:
```bash
npm run dev
```

### 2. Сборка для Android

#### Вариант А: Автоматическая сборка (рекомендуется)

Из корневой папки `kvantoriym-client`:

**Windows:**
```cmd
build-android.bat
```

**Linux/Mac:**
```bash
chmod +x build-android.sh
./build-android.sh
```

#### Вариант Б: Ручная сборка

```bash
cd web
npm run build
cd ..
# Скопируйте содержимое web/dist/ в android/app/src/main/assets/
```

### 3. Открытие проекта в Android Studio

1. Запустите Android Studio
2. File → Open → выберите папку `kvantoriym-client/android`
3. Дождитесь синхронизации Gradle
4. Подключите Android устройство или запустите эмулятор

### 4. Настройка для разработки (опционально)

Если хотите тестировать на устройстве с подключением к локальному серверу:

1. Узнайте IP адрес вашего компьютера в локальной сети
2. В `MainActivity.java` измените:
   ```java
   webView.loadUrl("http://ВАШ_IP:3000");
   ```
3. Убедитесь, что сервер доступен по этому адресу

### 5. Сборка APK

1. В Android Studio: Build → Build Bundle(s) / APK(s) → Build APK(s)
2. APK будет в `android/app/build/outputs/apk/debug/app-debug.apk`

### 6. Установка на устройство

```bash
adb install android/app/build/outputs/apk/debug/app-debug.apk
```

Или просто скопируйте APK на устройство и установите вручную.

## Настройка API для продакшена

Для продакшена нужно указать URL вашего сервера:

1. В `web/.env` укажите:
   ```
   VITE_API_URL=https://your-server.com/api
   ```

2. Пересоберите веб-приложение:
   ```bash
   cd web
   npm run build
   ```

3. Скопируйте файлы в Android проект (см. шаг 2)

## Разрешения

Приложение требует следующие разрешения:
- Интернет (для API запросов)
- Камера (для QR-сканера)

Эти разрешения уже настроены в `AndroidManifest.xml`.

## Устранение неполадок

### Приложение не загружается
- Проверьте, что файлы скопированы в `android/app/src/main/assets/`
- Проверьте логи в Android Studio (Logcat)

### QR-сканер не работает
- Убедитесь, что разрешён доступ к камере
- Проверьте, что библиотека html5-qrcode загружается

### API запросы не работают
- Проверьте URL API в конфигурации
- Убедитесь, что сервер доступен
- Проверьте CORS настройки сервера
